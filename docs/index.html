Features

🗂️

Deck Management

Browse and navigate all your AnkiDroid decks with a collapsible hierarchical tree view.

✨

AI Content Generation

Use Google Gemini to auto-generate explanations and content for your flashcard fields.

🌙

Automatic theme switching based on your system preferences for comfortable studying anytime.

🔒

Your Data, Your Key

Bring your own Gemini API key — stored locally on your device, never sent to our servers.

Requirements

AnkiFlow AI communicates with AnkiDroid via AnkiConnect Android. Both apps must be installed and AnkiConnect Android must have its service running before you launch AnkiFlow AI.

Setup Guide

  1. Install AnkiDroid Download AnkiDroid from Google Play and set up your decks.
  2. Install AnkiConnect Android Download the latest .apk from the GitHub Releases page or via the IzzyOnDroid F-Droid repo.
  3. Start AnkiConnect Android service Open the app, accept the permissions prompt, and tap Start Service. Keep it running in the background.
  4. Get a free Gemini API key Visit aistudio.google.com and create a free API key.
  5. Install AnkiFlow AI & enter your key Open AnkiFlow AI → Settings → paste your Gemini API key → Save.
  6. Start learning! Browse your decks, tap any note, and use the AI button to generate content for your flashcard fields.
